Public function Set_AreWebViewScriptApisEnabledForServiceWorkers(value: Integer): HResult; stdcall;

Enables or disables webview2 specific Service Worker JS APIs in the WebView2s associated with this Profile. When set to `TRUE`, chrome and webview objects are available in Service Workers. chrome.webview exposes APIs to interact with the WebView from Service Workers. The default value is `FALSE`. This setting applies to all newly installed Service Workers within the profile and is not persisted across WebView2 sessions. \snippet ScenarioServiceWorkerPostMessageSetting.cpp AreWebViewScriptApisEnabledForServiceWorkers
